### Backend Setup

1. **Navigate to project directory:**
   ```bash
   cd backend-assignment
   ```

2. **Install dependencies:**
   ```bash
   npm install

3. **Start the backend server:**
   ```bash
   npm start
   ```

4. **Backend should be running on:** `http://localhost:5000`

### Frontend Setup

1. **Navigate to frontend directory:**
   ```bash
   cd frontend
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the frontend server:**
   ```bash
   npm start
   ```

4. **Frontend should open automatically at:** `http://localhost:3000`
